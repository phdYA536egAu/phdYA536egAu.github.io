Recolha de dados e privacidade

O website pontogaming.pt é um projeto pequeno feito apenas por uma pessoa. Neste website, usando o clicky.com, são recolhidos dados para contabilizar o número de utilizadores do site, dados estes que são vitais para o funcionamento do site pois servem para perceber se o servidor PAGO que uso é capaz de aguentar a quantidade de utilizadores que frequentam este website GRÁTIS.

Os únicos dados pessoais recolhidos(pela clicky) são um UID gerado aleatoriamente e guardado num cookie. Não pretendo usar as estatísticas de utilização deste projeto para mais nada a não ser o propósito já indicado: manter o pontogaming.pt online de modo a que todos os utilizadores possam usufruir das minhas ferramentas!

Mais ainda, de acordo com o artigo 6 da GDPR, não é necessário pedir autorização ao utilizador para guardar o cookie acima indicado pois é "necessário para o funcionamento legítimo" do pontogaming.pt
